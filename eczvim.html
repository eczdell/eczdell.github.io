Quick Installation

Get started with my personal Neovim configuration:

# Backup existing Neovim config mv ~/.config/nvim ~/.config/nvim.backup 2>/dev/null || true # Clone eczvim git clone git@github.com:eczdell/eczvim.git ~/.config/nvim # Launch Neovim (plugins will auto-install) nvim

Requirements: Neovim 0.8+, Git, Node.js (for some LSP servers), and a Nerd Font

Essential Key Mappings

<leader>w Save file
<leader>e Toggle file explorer
<leader>f Find files (Telescope)
gd Go to definition
Tab Next buffer
S-Tab Previous buffer
jk Exit insert mode

Go Development

<leader>tt Run all tests
<leader>tn Run nearest test
<leader>tc Toggle test coverage
